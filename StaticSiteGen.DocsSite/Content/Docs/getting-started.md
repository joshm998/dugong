## What is StaticSiteGen
StaticSiteGen is a Framework built completely in .NET to create modern fast static websites.

## Key Features
- **Familiar:** Using Razor and Markdown make it easy to get started.
- **Provider Agnostic:** Works with S3, Azure Storage, CloudFlare, GitHub, Netlify, etc...
- **Fast:** The templates and most websites created achieve a LightHouse score above 99
- **Low Cost:** Designed to create websites that are low cost
